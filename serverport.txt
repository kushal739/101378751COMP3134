Q-1. What is the definition of server port?
Ans- A server port is a logical construct that refers to a specific communication endpoint or channel in a computer network. It is used to identify a particular service or process to which data or requests can be directed. 

Q-2. What is server port range?
Ans- Port numbers range from 0 to 65536, but only ports numbers 0 to 1024 are reserved for privileged services and designated as well-known ports.
